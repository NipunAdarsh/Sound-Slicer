# 🎶 Voice Separator

A Python project that separates vocals and instrumentals from songs using [Spleeter](https://github.com/deezer/spleeter).  
Built as a hands‑on learning project to explore audio processing, APIs, and Python environments.

---

## 🚀 Features
- Split any song into **vocals** and **instrumental** tracks
- Beginner‑friendly setup and usage
- Uses **TensorFlow** + **Spleeter** under the hood
- Ready for future features like **recording your own vocals** and mixing them into karaoke tracks
